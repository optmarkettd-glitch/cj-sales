# Фотографии команды

Поместите фотографии партнёров в эту папку:

1. **vladimir-chernov.jpg** - фотография Владимира Чернова (левая карточка)
2. **irina-zhirnyakova.jpg** - фотография Ирины Жирняковой (правая карточка)

## Требования к изображениям:

- Формат: JPG или PNG
- Рекомендуемый размер: минимум 400x400px (квадратные)
- Оптимальный размер: 600x600px
- Файлы должны быть оптимизированы для веба

## Как добавить:

1. Сохраните фотографии с указанными именами
2. Поместите их в папку `public/images/`
3. Изображения автоматически появятся на сайте
